# Stylus Ink Bench

Benchmark the Ink usage of a Stylus transaction

```
Usage: stylus-ink-bench [OPTIONS] --key <KEY> --program <PROGRAM> --signature <SIGNATURE>

Options:
  -r, --rpc <RPC>              Ethereum Client RPC [default: http://localhost:8547]
  -k, --key <KEY>              Private key of the sender
  -p, --program <PROGRAM>      Address of the Stylus program
  -s, --signature <SIGNATURE>  Function signature
  -a, --args <ARGS>            Function arguments
  -h, --help                   Print help
```
